# EmoVision-PyQt
## 使用须知
### 由于GitHub平台大小限制，文本分类模型和图像分类模型已放在HuggingFace中，若有需要，请到以下链接下载
[模型下载](https://huggingface.co/rrealGabe/EmoVision_model)

## 框架介绍

本应用采用了PyQt框架，其为跨平台开发的框架，可以实现一次部署，到处使用的功效。因此，本应用适合在各种平台上进行使用。
